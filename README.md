# Apple.in Clone
A clone of Apple.in built from scratch using pure HTML & CSS. Zero frameworks, zero libraries.

## 🛠️ Technologies Used
- HTML
- CSS

## 📄 Pages & Sections Built
- Sticky navigation bar with Apple logo, search and bag icons
- Community Letter from Tim (hero banner)
- iPhone section with Learn More & Shop iPhone buttons
- MacBook Neo section
- iPad Air M4 section
- Apple Watch Series 11 section
- AirPods Pro 3 section
- iPad Pro section
- iPhone Trade-in section
- Two-column product grid layout

## 📌 Note
All product images are referenced from Apple's official CDN for 
accuracy. This project is for learning purposes only and is not 
affiliated with Apple Inc.
